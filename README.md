# Coinstack
Coinstack is an open-source project for working with cryptographic keys and blockchain accounts, aimed at developers and researchers.
The project was created with an emphasis on transparency, simplicity of architecture, and user control.

Coinstack provides a convenient set of tools for managing keys, addresses, and basic operations without being tied to specific services, networks, or providers.
